# Spring Boot Patient API Challenge

## Task Overview
The goal is to create a Spring Boot API that fetches patient data from an external API and handles pagination.

- Create an endpoint `GET /api/patients` that accepts `page` and `size` parameters for pagination.
- Fetch data from an external API and return a paginated list of patients (you can simulate the data using a mock API or static data).
- Ensure the pagination works correctly and returns the appropriate data based on the `page` and `size` parameters.

**Time limit:** 90 minutes
